# Screenshots for the store listing

Capture 3–5 PNGs and upload them in the Overwolf Developer Console (they are not bundled in the .opk).

Suggested shots:
1. **Desktop dashboard** — an active session with several matches in the table.
2. **In-game overlay** — the compact widget over an Apex match (RP, last games, rank-up bar).
3. **Leaderboard** — the global leaderboard panel open (24h view).
4. **Rank-up progress** — overlay near a rank-up / Predator cutoff.

Recommended size: 1280×720 or larger, PNG.
