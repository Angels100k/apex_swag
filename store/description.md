# Apex Swag RP Tracker — Store Listing

**Category:** Apex Legends / Utility & Stats
**Supported game:** Apex Legends (Overwolf game id `21566`)

## Short description (one-liner)
Track your Apex ranked RP live — session gains, last games, rank-up progress, cloud history, and a global leaderboard.

## Long description
Apex Swag RP Tracker shows exactly how your ranked grind is going, in real time, without alt-tabbing.

**In-game overlay**
- Current RP and rank badge
- RP gained/lost this session
- Your last 4 games at a glance
- Progress bar to your next major rank-up (and the live Predator cutoff at the top)
- Toggle the overlay any time with a hotkey (default **Ctrl+Shift+A**)

**Desktop dashboard**
- Full match log for the current session
- Past sessions, synced from the cloud — survives reinstalls and works across PCs
- Global leaderboard: top RP gains across all players (24h / 7d / all-time)

**How it works**
Enter your Apex player name and platform once. The app reads match start/end from Overwolf game events and looks up your RP automatically — no manual entry needed. No separate program to run, no API key to register.

## Feature bullets (for the listing)
- Live in-game RP & rank overlay
- Automatic per-match RP tracking
- Cloud match history (cross-device)
- Global leaderboard
- Rank-up progress with live Predator cutoff
- Lightweight, draggable, hotkey-toggle overlay
